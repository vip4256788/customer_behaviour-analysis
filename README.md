# customer_behaviour-analysis


## 1️⃣ Project Overview

This project demonstrates an **end-to-end data analytics workflow**. The goal was to analyze a dataset, clean and explore the data, run analytical SQL queries, and build an interactive dashboard to present insights.

The project follows a real-world analytics process used by data analysts: **data loading → cleaning → exploration → SQL analysis → visualization → reporting**.

---

## 2️⃣ Dataset

The dataset used in this project contains structured data that was analyzed to identify patterns, trends, and useful insights.

Dataset tasks included:

* Loading the dataset into Python
* Cleaning missing or inconsistent values
* Preparing the data for analysis
* Storing the dataset in a PostgreSQL database for SQL queries

---

## 3️⃣ Tools & Technologies

The following tools were used in this project:

* **Python** – Data loading, cleaning, and exploratory data analysis
* **Pandas & NumPy** – Data manipulation and preprocessing
* **Matplotlib / Seaborn** – Data visualization during EDA
* **PostgreSQL** – Database storage and SQL analysis
* **SQL** – Writing queries to analyze the dataset
* **Power BI** – Creating an interactive dashboard
* **Gamma AI** – Generating the final presentation (PPT)

---

## 4️⃣ Project Steps

### Step 1: Data Loading

The dataset was loaded into Python using Pandas for initial inspection.

### Step 2: Data Cleaning

Data preprocessing included:

* Handling missing values
* Removing duplicates
* Correcting data types
* Standardizing column names

### Step 3: Exploratory Data Analysis (EDA)

EDA was performed to understand the dataset and identify trends using statistical summaries and visualizations.

### Step 4: SQL Analysis

The cleaned data was stored in a **PostgreSQL database**, and SQL queries were used to:

* Perform aggregations
* Filter and group data
* Extract meaningful insights

### Step 5: Dashboard Creation

An interactive **Power BI dashboard** was built to visualize important metrics and trends from the dataset.

### Step 6: Reporting & Presentation

The findings were summarized in a **data analysis report**, and a **presentation was created using Gamma AI** to clearly communicate insights.

---

## 5️⃣ Dashboard

The Power BI dashboard includes:

* Key performance metrics
* Data trends and patterns
* Interactive charts and filters
* Visual insights derived from the dataset

The dashboard helps stakeholders quickly understand the most important insights from the data.

## 6️⃣ Results & Insights

Key outcomes of this project:

This project demonstrates practical skills in **data cleaning, analysis, SQL querying, and dashboard creation**.

## 7️⃣ How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/data-analytics-project.git
```

2. Install required Python libraries

```
pip install pandas numpy matplotlib seaborn psycopg2
```

3. Run the Python notebook/script to perform data cleaning and EDA.

4. Load the cleaned dataset into **PostgreSQL** and run SQL queries.

5. Open the **Power BI (.pbix)** file to view the dashboard.
